## ✨ Features  

- 📖 **קריאת התנ״ך בעברית מלאה** — גישה לכל ספרי התורה, הנביאים והכתובים.  
- 🔍 **בחירת ספר ופרק** — ניווט פשוט וברור בין ספרים ופרקים.  
- 📱 **PWA מלאה** — ניתן להתקין את האפליקציה על המסך הראשי ולהשתמש בה גם ללא חיבור לאינטרנט.  
- 🕮 **עיצוב מינימליסטי ונקי** — חוויית קריאה נעימה ונטולת הסחות דעת.  
- ⚙️ **טעינה מהירה** — הנתונים נטענים בזמן אמת מה־Sefaria API.  

---

## 🚀 Getting Started  

1. **ביקור באפליקציה:**  
   פתח את האתר בדפדפן שלך:  
   [תנ״ך – Tanahk PWA](https://yardenfalik.github.io/tanahkApp/)  

2. **הוספה למסך הבית:**  
   - **במובייל:** פתח את תפריט הדפדפן ובחר „הוסף למסך הבית“.  
   - **בדסקטופ:** השתמש בהצעת ההתקנה שמופיעה בדפדפן כדי להתקין את האפליקציה.  

3. **קריאה:**  
   בחר ספר ופרק — והתחל לקרוא ישירות מהדפדפן שלך.  

---

## 🛠️ Usage  

1. **בחירת ספר:**  
   בחר מתוך רשימת ספרי התנ״ך המוצגים במסך הראשי.  

2. **ניווט בפרקים:**  
   השתמש בכפתורים „פרק קודם“ ו„פרק הבא“ כדי לעבור בין פרקים.  

3. **קרדיט חוקי:**  
   כל הטקסטים באפליקציה מסופקים על־ידי [Sefaria](https://www.sefaria.org).  
   שימוש זה מותר תחת רישיון שימוש לא מסחרי.
